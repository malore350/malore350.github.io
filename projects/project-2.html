Platform Experience

Technical Architecture

Built as a high-security full-stack system. The backend handles blockchain integrations and data persistence, while the frontend provides a responsive trading UX.

Core Stack
  • Backend: Node.js + Express.js (JWT auth)
  • Database: Dual-support for MongoDB & MySQL
  • Web: Next.js 15 (App Router) + shadcn/ui
  • Mobile: React Native + Expo SDK 54
Security & Operations
  • Authentication: 2FA (Speakeasy) + Secondary PIN
  • Price Data: Binance API via secure proxy
  • Infrastructure: node-cron + Mailgun integration
  • Privacy: Tor network hosting & censorship resistance
