Gaming meets artistry. This website offers a thoughtfully curated selection of both classic and new games, emphasizing stunning visuals and elegant design.

Contains 7 separate games with a focus on aesthetics. All games designed from scratch. Some games inspired by NYT Games. Built on Next.js + TailwindCSS.

Project Overview & Technical Details

GameHub is a client-heavy Next.js 15 application that aggregates classic puzzle and logic games, with gameplay, validation, and timing executed entirely in the browser. The architecture emphasizes fast iteration, UI responsiveness, and reusable game logic modules without relying on server-side APIs.

  • Framework: Next.js 15 (App Router) with React 18; Turbopack enabled for dev builds.
  • Language: TypeScript in strict mode across the app and game logic.
  • UI/Styling: Tailwind CSS v3.3, Framer Motion animations, Radix UI primitives (via Shadcn UI).
  • State: React Hooks only (useState/useReducer/custom hooks), no global state libraries.
  • Icons: @mui/icons-material and Lucide React.
  • Structure: app/ for routes, components/ for reusable UI, lib/ for puzzle generators, hooks/ for gameplay state.
  • Features: Sudoku, Minesweeper, 2048, Numble, Chemle, and word games like Wordle-style and Spelling Bee.
  • Theming: next-themes with per-game style presets (e.g., Classic vs Royal).
  • Responsive UX: Mobile-first controls and game-specific input patterns (e.g., numeric pads for Sudoku).
