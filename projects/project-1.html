Technical Deep Dive

GameHub is a client-heavy Next.js 15 application that aggregates classic puzzle and logic games. The architecture emphasizes fast iteration, UI responsiveness, and reusable game logic modules.

Core Stack
  • Framework: Next.js 15 (App Router) + React 18
  • Styling: Tailwind CSS + Radix UI Primitives
  • Animations: Framer Motion for fluid state transitions
  • Logic: Strict TypeScript with client-side puzzle generation
Key Features
  • 7+ self-contained logic and word games
  • Dynamic dark/light mode with game-specific themes
  • Mobile-first controls with specialized input pads
  • Zero-dependency internal state management
