A comprehensive university consultancy platform designed to help students find universities, manage applications, and schedule consultations via a modern monorepo architecture.

React Node.js Supabase React Native TypeScript

Platform Interface

Technical Architecture

Designed for high scalability and real-time interaction across web, mobile, and backend services.

Backend & Infrastructure
  • Server: Node.js + Express.js
  • Database: Supabase (PostgreSQL)
  • Auth: Hybrid Supabase + Custom JWT
  • Storage: Supabase S3 Storage
Frontend Ecosystem
  • Web: React (Vite) + Tailwind + Shadcn UI
  • Mobile: React Native + Expo SDK 54
  • State: Zustand (Web) & Context (Mobile)
  • i18n: Multi-language (i18next)

User Roles & Workflows

Student

Search universities via AI, manage applications with credits, upload documents, and schedule expert consultations.

Employee

Guide assigned students, manage application lifecycles, and handle real-time support and consultations.

Admin

Full operations control: user management, staff assignments,promo code tracking, and financial oversight.

Financial Ecosystem

Payment Gateway

Integration with Epoint (Azerbaijan) for secure transaction signing, RRN tracking, and automated invoicing.

Monetization

Tiered subscription plans (LITE, PRO, MAX) bundling application credits, AI suggestions, and priority support.
