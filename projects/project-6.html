CryptoPro Mobile

React Native cryptocurrency exchange app.

Mobile App
Part of CryptoPro Platform

A production-ready React Native (Expo) cryptocurrency exchange mobile application featuring wallet management, market data, token swaps, and liquidity pool features with biometric authentication.

React Native 0.81 Expo SDK 54 TypeScript React Navigation v7 Axios Reanimated 4

App Screens

Technical Architecture

Built with Clean Architecture using React Context for state management. The app connects to the CryptoPro backend API with secure JWT authentication.

Core Stack
  • Framework: React Native 0.81.4 + Expo SDK 54
  • Language: TypeScript ~5.9.2
  • Navigation: React Navigation v7 (Native Stack + Bottom Tabs)
  • State: React Context API
  • HTTP: Axios with JWT interceptors
Storage & Security
  • Storage: AsyncStorage + SecureStore
  • Biometrics: expo-local-authentication (Face ID/Fingerprint)
  • Animations: React Native Reanimated 4.1
  • UI Effects: Expo Blur, Expo Linear Gradient
  • Forms: React Hook Form + Zod validation
Navigation Structure
Navigator Type Screens
Root Auth Check Routes to Auth or Main navigator based on login state
Auth Stack Login, Signup screens
Main Bottom Tabs Assets, Markets, Swap, Earn, Profile
Markets Nested Stack Market list, Coin details

Key Features

JWT Authentication

Secure token-based login with automatic token refresh via axios interceptors.

Biometric Login

Face ID / Fingerprint authentication with credentials stored in SecureStore.

Portfolio Tracking

View cryptocurrency holdings with icons and real-time values.

Token Swap

Swap interface for exchanging tokens directly in the app.

Earn & Liquidity

Staking and liquidity pool contribution/withdrawal features.

Profile & Settings

User settings, security options, PIN management, and support tickets.

API Integration

The mobile app connects to the CryptoPro backend API with the following configuration:

Setting Value
Base URL Production: Tor onion service / Dev: localhost:5000
Timeout 10 seconds
Auth Header Bearer token in Authorization header
Token Storage AsyncStorage with SecureStore for biometric credentials
Dev URLs localhost:5000 (iOS), 10.0.2.2:5000 (Android)
